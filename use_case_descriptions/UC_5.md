## 設定をプリセットとして登録

| タイトル | 内容 |
| --- | --- |
| ユースケース名 | 設定をプリセットとして登録 |
| 目的 | ゲームを設定するときによく使う設定を使えるようにするため |
| アクター | ユーザー |
| 画面 | ゲーム開始画面 |
| 前提条件 | UC_6、UC_7、UC_8が済んでいること |
| メインフロー | 1. ユーザーはプリセット登録ボタンを押す<br>2. システムは入力されているデータを保存する<br>3. システムは「プリセットを登録しました」と画面に表示させる |
| 代替フロー |  |
| 備考 | ユーザーは未入力でも可能 |
